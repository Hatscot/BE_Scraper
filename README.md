# BE_Scraper
Das Skript Scraper.py soll ermöglichen alle Sets zu Scrapen und in eine Liste zu Packen die Liste soll folgender Maßen ausschauen:
| Set Gruppe | Set Nummer | Set Name mit eingebetteten Link | Set Jahr | Set preis - Value | Growth in Prozent |

Das Skript Pic_Scraper soll ermöglichen das:
Alle Bilder anhand einer Vorgefertigten Liste die über die Datei "scraper.py" schon erstellt worden ist alle Bilder der Jeweiligen sets zu Ziehen mit Folgender Ordner Strucktur:
[Pics]
  [(Set Nummer)]
    (Set Name) xx.png/.jpg - Eines von Beiden {xx Steht für die Reihenfolge Sprich Bild 01 - 02 ...}

Es soll bei jeder Set Nummer ein anderen Ordner angelegt werden damit alles Steril Bleibt dabei steht [] für Ordner und () für den Variabel Namen der in der eingespeisten Liste gesucht wird
